# DEL PINO Knowledge Agent

Agente interno de conocimiento que responde consultas sobre políticas comerciales y operativas de DEL PINO home & deco utilizando recuperación aumentada por generación.

**Estado:** MVP en desarrollo.

## Stack previsto

- Python 3.12
- LangChain
- Google Gemini
- PyPDF
- FAISS
- Streamlit
- python-dotenv

> Este proyecto utiliza documentación ficticia preparada con fines educativos para el Challenge Final Tech AI Builder.
